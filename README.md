# Adoguard APP
Adoguard is a high-performance credential manager built for speed and security. It utilizes a "zero-local-footprint" policy to ensure your sensitive data remains private and ephemeral.

# Core Functionality
Vault Management: Securely store usernames, site URLs, and passwords.

Zero-Trace Security: No data is permanently stored on the local machine; info stays in your control.

Entropy Engine: Integrated generator for creating complex, high-strength passwords.

Workspaces: Support for multiple isolated profiles to categorize your credentials.
